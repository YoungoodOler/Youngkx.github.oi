我的OI之路

旅程

我只是学了几个月，但因为各种原因被动退役了，主要是学业和家长老师的原因。好像没有一个人支持我搞OI，也是因为太多时间花在OI上，我的文化课太差，所以他们都不支持。其实这两者没有必然的联系，可能只是因为我不太努力吧，于是两边卡着我，什么都搞不好。

但是他们都不懂我们搞OI是在干什么，我只是对OI有种热情而已，学习算法以及编写程序真的让我感到快乐，尤其是洛谷AC时的“恭喜你通过了此题！”。

以后到了大学参加ICPC吧……

之前看了一篇文章《如何优雅的和竞赛分手》，深有感触，但是那篇文章排版太烂，我强迫症犯了，于是重写了一篇在下面。

优雅的和竞赛说分手

原文链接（知乎）

退役的原因

  • 主动退役
    • 失去兴趣
    • 认为自己没有能力得奖，没必要继续
  • 被动退役
    • 学业原因，例如文化课成绩太差或上高三了
    • 长辈例如班主任或家长不支持

退役后的心路历程

主动退赛

  1. 后悔：极度想要回归竞赛。但不建议这样，一来二去耽误了很多时间，所以主动退赛一定要考虑清楚；
  2. 烦躁、留恋：时不时想起竞赛的日子，一想到就烦，还经常会拿出竞赛题目写；
  3. 接受、适应：总对竞赛还有些恋恋不舍，但这种感觉逐渐消失，慢慢地投入到了文化课的学习之中。最后渐渐不再会想到竞赛的事了；
  4. 回归文化课：全身心投入文化课学习，与大部分学生一样。

被动退赛

  1. 烦躁、留恋：此时人的情绪会和主动退赛很不同，因为不是自愿退出的，总会想着要回去，心情也很烦闷，被动退赛的大部分磨合时间都在这里面了；
  2. 接受、适应；
  3. 回归文化课：回归正常的学习生活；
  4. 回归竞赛：很多人其实等以后上了大学会继续参加大学生竞赛，尤其是学OI的，肯定有很多被动退赛的人跟我一样准备大学去搞ICPC吧。

调节方法

腹式呼吸法

腹式呼吸是让横膈膜上下移动。由于吸气时横膈膜会下降，把脏器挤到下方，因此肚子会膨胀，而非胸部膨胀。因此，吐气时横膈膜将会比平常上升，因而可以进行深度呼吸，吐出较多易停滞在肺底部的二氧化碳。由于其吸入的氧量高于正常情况下的两到三倍，同时将注意力转移到呼吸上，所以可以起到冷静情绪等作用。

  • 练习方法

    • 吸气：采取仰卧或舒适的坐姿，可以把一只手放在腹部肚脐处，放松全身，先自然呼吸，然后吸气，最大限度地向外扩张腹部，使腹部鼓起，胸部保持不动。
    • 呼气：胸部保持不动。最大限度地向内收缩腹部，把所有废气从肺部呼出去。保持每一次呼吸的节奏一致，细心体会腹部的一起一落。

  • 注意

    • 呼吸要深长而缓慢，用鼻吸气用口呼气。
    • 一呼一吸掌握在15秒钟左右。即深吸气（鼓起肚子）35秒，屏息1秒，然后慢呼气（回缩肚子）68秒，屏息1秒。

注意力转移

  • 目标转移法：如果你在工作和生活中因为某个人或者某一件事情感到心情烦躁、郁闷，而注意力又无法集中，这时你就不要强迫自己了，应该换一种方式放松一下烦躁郁闷的心情，最好的方法就是听听音乐，看看电视，上上网，这样做就会分散一下紧张的注意力，让烦躁郁闷的心情能够及时得到缓解和放松。

  • 心理暗示法：遇到心情烦躁不愉快时，应该采用积极的暗示方法，也就是用积极的心态去思考，告诉自己心情烦躁是暂时的，属于正常心里反应，乌云就会散去，温暖的阳光即将到来。还可以回忆从前所发生在自己身上的美好事情，这样做就会缓解由心里压力造成的心情烦躁。

  • 思想交流法：找一位比较知心的朋友，把自己的不愉快跟朋友倾诉一下，讲讲自己的心里的真实感受。这样做你就会把不愉快和烦躁的心情释放出去，随之而来的就会是心情舒畅了，烦躁和不安的心里也得到了解决，何乐而不为呢。

  • 改变环境法：到自己想去的地方，去能够让人心旷神怡的美丽的地方；去见想见的人，用快乐抵消烦恼；改变生活节奏，迎接新自我。

蝴蝶拍

  • 练习方法

    1. 轻闭双眼，调节呼吸，感受自然。
    2. 双臂在胸前交叉，右手在左侧、左手在右侧，轻抱自己对侧的肩膀。
    3. 双手轮流轻拍自己的臂膀，左一下、右一下为一轮
    4. 速度要慢，轻拍 4—12轮为一组。停下来，深吸一口气，感受一下，自己的感觉如何？
    5. 此时大脑里会浮现很多画面，这是人的正常情感流露，不要去干预它们。
    6. 再来做一组蝴蝶拍。

写在最后

人生的路还很长，通向成功的路也不止一条，该来的总会来的，不必急于这一时。
既然目标是地平线，留给世界的只能是背影。

我的座右铭：我见青山多妩媚，料青山见我应如是。

加油吧，少年！
